Message driven PHP

Message based domain layers. Shipped with industry standard infrastructure. Star

MsgPHP is a project that aims to provide (common) message based domain layers for your application. It has a low development time overhead and avoids being overly opinionated.

  • ↘ A base Domain to rapidly built other domains
  • ↘ User management with the User domain
  • ↘ Entity attribute values with the Eav domain
